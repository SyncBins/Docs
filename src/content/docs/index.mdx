## Quick look

SyncBins organises your content into **bins** — named, emoji-tagged collections with a configurable capacity. Every item you add is encrypted on-device before it leaves, pushed to the server, and fanned out to your other devices in real time.

The hosted service lives at **[syncbins.com](https://syncbins.com)**. The server is open source — run it yourself on any machine that can run Docker.
