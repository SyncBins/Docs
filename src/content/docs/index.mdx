<h2 class="sb-section-h">What's in the box</h2>

<CardGrid>
  <Card title="End-to-end encrypted" icon="seti:lock">
    XChaCha20-Poly1305 + Curve25519. Server stores ciphertext only — never the master key.
  </Card>
  <Card title="Self-hostable" icon="rocket">
    Single Go binary or one Docker compose. BYO Azure Blob, S3, R2, or local disk.
  </Card>
  <Card title="Real-time sync" icon="bars">
    WebSocket fanout to every paired device. Drops are visible across devices in under 1s.
  </Card>
  <Card title="10 content types" icon="open-book">
    Passwords, links, code, voice memos, screenshots — each with a custom renderer.
  </Card>
  <Card title="Agent-ready" icon="seti:default">
    `@syncbins/mcp` exposes the same store to Cursor, Claude Desktop, and any MCP client.
  </Card>
  <Card title="Single user, by design" icon="seti:plan">
    No teams, no sharing, no permission models. Future-you is the only audience.
  </Card>
</CardGrid>
