Packaged desktop apps (likely Tauri or Electron) would give you a menubar drop target, OS keychain integration, and offline-tolerant sync — same server, same crypto.

## Status today

**Not available.** There is no desktop repo or release artifact yet. SyncBins runs in the browser and via [`@syncbins/mcp`](/integrations/mcp/) for agent workflows.

## What works instead (now)

| Need | Use |
|------|-----|
| Daily use on laptop | Web app — pin the tab, set a [quick-unlock PIN](/get-started/quick-unlock/) |
| Self-hosted server | [Docker quick start](/self-hosting/quickstart/) |
| Cursor / Claude access | [MCP server](/integrations/mcp/) |

<Aside type="note" title="Living doc">
Desktop docs will appear when there is a downloadable build and a pairing flow implemented in code — not before.
</Aside>
