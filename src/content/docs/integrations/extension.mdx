A SyncBins browser extension is planned — quick capture from the toolbar, pair as a device, same encryption envelope as the web app.

## Status today

**Not available.** The [Extension](https://github.com/SyncBins/Extension) repository exists but contains no extension code yet. There is nothing to install.

## What works instead (now)

Use the web app at [syncbins.com](https://syncbins.com) or your self-hosted URL:

- Paste URLs or text in the [composer](/get-started/first-item/)
- Use the clipboard hint when copying on the same machine
- Pair the browser as a device ([Pairing more devices](/get-started/pairing/))

For agents, use the [MCP server](/integrations/mcp/) — that **is** shipping today.

<Aside type="note" title="Living doc">
When the extension reaches a testable build, this page will document install steps, permissions, and pairing. Until then we won't speculate on manifest v3 details or store listings.
